# FutureLight Figma Page Generator

This is a free local Figma plugin that generates the FutureLight page map as mobile H5/Android frames inside the current Figma page. It is designed for Figma Free plan by using one canvas page only.

## Use

1. Open Figma Desktop.
2. Create or open a file named `FutureLight App Design`.
3. Go to `Plugins -> Development -> Import plugin from manifest...`.
4. Select `tools/figma-futurelight-pages/manifest.json`.
5. Run `FutureLight Page Generator`.

## Output

- One Figma page renamed `FutureLight - Free Plan Single Canvas`.
- A Sitemap section on the same canvas.
- A Components section on the same canvas.
- 79 mobile frames on the same canvas.

The generated frames are structured mobile wireframe/design frames, not final production visual art.
