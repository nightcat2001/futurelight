# Accessibility Report

- Result: PASS
- Touch Target Rule: All interactive controls are at least 44px high; primary child controls are at least 64px.
- Typography Rule: Parent UI uses 16px minimum body text; child-facing labels pair short text with large visual cues.
- Reduced Motion Rule: Short purposeful transitions under 260ms; bedtime mode removes bounce and loops.

Per-page accessibility notes:

- FL-HTML-01: Single H1, CTA first after explanation, contrast above AA, touch target 56px.
- FL-HTML-02: Labels are explicit; errors use aria-live; tab order follows form order.
- FL-HTML-03: Avatar choices have text labels; nickname field has helper text.
- FL-HTML-04: Cards are radio buttons; selected state has text and border.
- FL-HTML-05: Recommendation reasons are text, not only color chips.
- FL-HTML-06: Audio captions available to parent; controls have labels.
- FL-HTML-07: Prompt can be read aloud; visual answers include labels.
- FL-HTML-08: Time and calmness are text; not color-only.
- FL-HTML-09: Steps are ordered text; timer does not rely on animation.
- FL-HTML-10: Parent instructions and child choices are separated semantically.
- FL-HTML-11: Charts include text summaries and table fallback.
- FL-HTML-12: High contrast in dark mode; audio can continue with screen dimmed.
