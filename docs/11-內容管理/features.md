# 內容管理功能設計

## 功能 1：課程管理

管理員可建立、編輯、發布與下架課程。

課程狀態：

- 草稿。
- 已發布。
- 已下架。

## 功能 2：單元管理

管理員可在課程底下建立單元。

規則：

- 單元需有排序。
- 單元可獨立設定是否啟用。
- 已發布課程修改單元時需注意前台學習進度。

## 功能 3：學習內容管理

管理員可建立單字、句子、故事段落與聽力內容。

內容資料：

- 標題。
- 文字。
- 翻譯。
- 音檔。
- 圖片。
- 內容類型。

## 功能 4：題庫管理

管理員可為單元建立練習題。

題型：

- 單選。
- 圖片選擇。
- 聽音選答案。
- 配對。
- 排序。

## 功能 5：媒體管理

管理員可上傳與管理圖片、音檔。

規則：

- 媒體需有檔案類型與大小限制。
- 媒體可被多個內容引用。
- 刪除媒體前需檢查是否被使用。

## 功能 6：稽核紀錄

重要內容操作需留下紀錄。

紀錄項目：

- 操作者。
- 操作類型。
- 操作目標。
- 操作時間。
