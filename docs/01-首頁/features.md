# 首頁功能設計

## 功能 1：依登入狀態顯示入口

未登入使用者進入首頁時，顯示登入、註冊與公開課程探索入口。

規則：

- 未登入不可看到孩子姓名、進度、推薦內容。
- 已登入但沒有孩子資料時，主要行動是建立孩子資料。
- 已登入且有孩子資料時，顯示個人化學習首頁。

## 功能 2：今日學習推薦

首頁顯示最適合目前孩子今天開始的課程、單元或練習。

推薦優先順序：

1. 尚未完成且最近學過的單元。
2. 已完成學習但尚未練習的單元。
3. 符合孩子程度的新課程。
4. 沒有足夠資料時顯示熱門入門課程。

## 功能 3：快速開始

孩子或家長點擊快速開始後，系統自動導向最合適的學習位置。

導向規則：

- 有未完成單元時，導向該單元的學習播放頁。
- 單元已學完但練習未完成時，導向練習遊戲頁。
- 沒有任何進度時，導向推薦課程的第一個單元。

## 功能 4：學習摘要

顯示目前孩子的學習狀態，讓家長與孩子能快速感受到進度。

顯示資料：

- 今日是否已學習。
- 連續學習天數。
- 已完成單元數。
- 本週練習答對率。
- 已獲得星星或點數。

## 功能 5：孩子切換

家長有多位孩子時，可在首頁切換目前孩子。

規則：

- 切換後首頁推薦、學習摘要、快速開始目標都要同步更新。
- 使用者偏好可記住上次選擇的孩子。
