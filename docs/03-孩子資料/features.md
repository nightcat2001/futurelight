# 孩子資料功能設計

## 功能 1：建立孩子資料

家長第一次使用時，需要建立至少一位孩子。

必填資料：

- 暱稱。
- 年齡或生日。
- 目標語言。
- 目前程度。

## 功能 2：編輯孩子資料

家長可更新孩子基本資料與語言設定。

規則：

- 修改目標語言後，推薦課程需重新計算。
- 修改程度後，課程探索預設篩選需更新。
- 已存在的學習紀錄不可因資料修改而消失。

## 功能 3：多孩子管理

同一家長帳號可管理多位孩子。

規則：

- 每位孩子有獨立學習進度。
- 每位孩子有獨立獎勵與成就。
- 切換孩子後所有學習頁面都以該孩子為主。

## 功能 4：學習偏好

家長可設定孩子偏好的學習方式。

可設定項目：

- 每次學習時間。
- 喜歡的主題。
- 是否偏好遊戲式練習。
- 是否需要更多聽力或發音內容。

## 功能 5：停用孩子資料

家長可停用孩子資料，但不直接刪除學習紀錄。

規則：

- 停用後不出現在孩子切換清單。
- 停用前需二次確認。
- 管理者可依資料政策保留或匯出紀錄。
