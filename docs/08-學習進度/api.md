# 學習進度 API

## 需要的 API

### GET /api/children/{child_id}/progress/summary

取得孩子學習總覽。

### GET /api/children/{child_id}/progress/courses

取得課程完成度。

### GET /api/children/{child_id}/progress/skills

取得能力分類統計。

### GET /api/children/{child_id}/learning-history

取得近期學習紀錄。

## 回傳需求

- 已完成課程數。
- 已完成單元數。
- 連續學習天數。
- 各能力分類分數或等級。
- 近期學習事件。
