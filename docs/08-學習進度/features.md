# 學習進度功能設計

## 功能 1：學習總覽

顯示孩子目前整體學習狀態。

資料：

- 已完成課程數。
- 已完成單元數。
- 本週學習天數。
- 連續學習天數。
- 總學習時間。

## 功能 2：課程進度

顯示孩子在每門課程的完成度。

規則：

- 完成度以完成單元數除以總單元數計算。
- 若課程下架，仍可在歷史紀錄中顯示。
- 點擊課程可進入課程詳情。

## 功能 3：能力分析

根據學習與答題結果顯示能力分類。

能力分類：

- 單字。
- 聽力。
- 發音。
- 閱讀。
- 句型。

## 功能 4：近期紀錄

列出孩子近期學習活動。

紀錄類型：

- 完成單元。
- 完成練習。
- 取得獎勵。
- 完成課程。

## 功能 5：加強建議

根據弱點分析推薦下一步。

規則：

- 答錯率高的主題優先。
- 太久未複習的內容優先。
- 推薦需能導向課程或練習。
