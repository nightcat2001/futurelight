# 系統設定功能設計

## 功能 1：帳號資料

使用者可修改基本帳號資料。

可修改資料：

- 顯示名稱。
- 介面語言。
- 時區。
- 預設孩子。

## 功能 2：密碼修改

使用者可修改登入密碼。

規則：

- 需輸入舊密碼。
- 新密碼需符合安全規則。
- 修改成功後可選擇讓其他裝置登出。

## 功能 3：通知設定

使用者可調整通知偏好。

通知類型：

- 學習提醒。
- 每週報告。
- 獎勵通知。
- 系統通知。

## 功能 4：隱私設定

使用者可查看與調整資料相關設定。

項目：

- 學習資料使用說明。
- 匯出孩子學習紀錄。
- 停用帳號申請。

## 功能 5：安全紀錄

使用者可查看近期安全事件。

事件類型：

- 登入。
- 登出。
- 密碼修改。
- 密碼重設。
- 重要設定變更。
