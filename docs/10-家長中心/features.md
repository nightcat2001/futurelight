# 家長中心功能設計

## 功能 1：孩子總覽

家長可查看所有孩子的摘要。

顯示資料：

- 孩子暱稱。
- 目前學習語言。
- 今日是否學習。
- 本週完成單元。
- 近期需要加強項目。

## 功能 2：學習目標設定

家長可為孩子設定學習目標。

目標類型：

- 每日學習分鐘數。
- 每週完成單元數。
- 每週練習次數。

## 功能 3：提醒設定

家長可設定學習提醒。

可設定項目：

- 是否開啟提醒。
- 提醒時間。
- 每週報告。
- 練習未完成提醒。

## 功能 4：孩子管理入口

家長可從家長中心進入孩子資料管理。

操作：

- 新增孩子。
- 編輯孩子。
- 切換孩子。
- 停用孩子資料。

## 功能 5：學習報告

家長可查看週報式學習摘要。

內容：

- 本週完成內容。
- 表現良好的能力。
- 需要加強能力。
- 建議下週學習方向。
