每次把 Todo 項目標記完成前，必須跑本文件的命令，或在交付紀錄中清楚寫明哪一項沒有跑、原因是什麼、替代驗證是什麼。本文件是 `P0-QA-001` 的主清單。

## One Command

在 repo root 執行：

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\tools\verify-delivery.ps1
```

若使用非預設資料庫：

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\tools\verify-delivery.ps1 -DatabaseUrl 'postgres://futurelight:futurelight@localhost:37432/futurelight'
```

Prerequisites:

- PostgreSQL 必須使用 assigned host port `37432`。
- 不得啟動舊 port `5173 / 4000 / 5433`。
- 若 backend dev server 正在執行，先停止再跑 Rust compile/test 類命令，避免 Windows 鎖住 `target\debug\backend.exe`。

## Content / Asset Rules

`tools/check-content-assets.mjs` 會檢查：

- `assets/asset_manifest.json` 必須是合法 JSON。
- 所有非 planned asset 必須有實際檔案路徑。
- asset id 不可重複。
- `content/courses/*.json` 必須包含 course id/title/status/units。
- 每個 unit 必須有 steps 和 practice。
- step 的 image asset 必須存在於 manifest。
- published course 不可缺 `audio_asset_id`。
- practice `correct_answer` 必須對應同單元 step text。

Draft course 可以暫時缺音檔，但不得把 draft 當成完成內容或上架內容。

## PostgreSQL Publish Rules

`cargo run -- check-content` 會檢查 PostgreSQL 裡所有 `status = 'published'` 的課程：

- published course 必須有 cover asset。
- published course 必須至少有一個 lesson。
- published lesson 必須至少有一個 activity。
- published activity 必須有 `answer_key.correct_answer` 或 `answer_key.expected`。
- published `word_card` 必須有 audio asset key，且必須有 image/color/counting material。
- published course/activity 連到的 asset 必須存在，且狀態必須是 `approved` 或 `published`。
- 未審核 AI 素材不可進 published course。

目前核心課程 seed 是 draft；這些規則會在任何人把課程改成 published 時擋住未完成素材與題目。

## Secret Scan Scope

`tools/check-secrets.ps1` 會排除：

- `.git`
- `frontend/node_modules`
- `frontend/dist`
- `backend/target`
- `docs/artifacts`
- lockfiles

它會掃描常見 OpenAI、GitHub、AWS、Google、Slack、Stripe、private key 與高風險 secret assignment pattern。掃到疑似 secret 時，不可直接提交；先確認是否為假陽性，必要時移除並輪換金鑰。

## Browser Verification

有 UI 或流程變更時，除上述命令外還需要真實瀏覽器驗證：

- 桌面 viewport smoke。
- 手機 viewport smoke。
- 主要 CTA 點擊流程。
- 登入、建立孩子、開始課程、提交練習、家長中心操作等被改到的流程。

瀏覽器驗證結果必須寫回 Todo 或交付說明。不能只用 build pass 代表 UI 完成。
