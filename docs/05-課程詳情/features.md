# 課程詳情功能設計

## 功能 1：課程資訊展示

顯示課程的完整介紹，讓家長知道孩子會學到什麼。

內容：

- 課程名稱。
- 課程描述。
- 學習目標。
- 適合年齡。
- 目標語言與程度。
- 預估完成時間。

## 功能 2：單元列表

顯示課程底下所有單元與孩子進度。

單元狀態：

- 尚未開始。
- 學習中。
- 已完成。
- 已完成練習。

## 功能 3：開始學習

孩子可從課程詳情開始第一個單元。

規則：

- 尚未開始課程時建立進度紀錄。
- 開始後導向第一個可學習單元。

## 功能 4：繼續學習

已開始課程時，主要按鈕顯示繼續學習。

導向規則：

- 有未完成學習步驟時，導向學習播放頁。
- 學習已完成但練習未完成時，導向練習遊戲頁。
- 課程全部完成時，顯示複習或重新練習入口。

## 功能 5：課程可見性

只有已發布課程可被一般使用者查看。

規則：

- 草稿課程只給管理員預覽。
- 下架課程不可從課程探索進入。
- 已開始但後來下架的課程需顯示不可繼續或替代課程提示。
