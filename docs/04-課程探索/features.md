# 課程探索功能設計

## 功能 1：課程列表

顯示所有可供前台使用者學習的已發布課程。

課程卡片資料：

- 課程名稱。
- 封面圖片。
- 目標語言。
- 程度。
- 適合年齡。
- 單元數。
- 孩子的完成狀態。

## 功能 2：搜尋課程

使用者可用關鍵字搜尋課程。

搜尋範圍：

- 課程名稱。
- 課程描述。
- 主題名稱。

## 功能 3：篩選課程

使用者可依孩子需求縮小課程範圍。

篩選條件：

- 目標語言。
- 程度。
- 年齡。
- 主題。
- 課程長度。

## 功能 4：推薦課程

系統依孩子資料顯示推薦課程。

推薦依據：

- 目標語言。
- 目前程度。
- 學習紀錄。
- 答題弱點。
- 家長設定的偏好主題。

## 功能 5：空狀態引導

當搜尋或篩選沒有結果時，提供可行下一步。

顯示方式：

- 提示放寬篩選條件。
- 提供清除搜尋按鈕。
- 顯示幾門熱門課程。
