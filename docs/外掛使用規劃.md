# Codex 外掛使用規劃

FutureLight 會把可用的 Codex 能力分成開發驗證、美術資源、內容品管與文件資料四條線。

## 已啟用工作流

| 能力 | 用途 | 目前狀態 |
| --- | --- | --- |
| Browser | 開 localhost、點按鈕、截圖、檢查互動 | 已納入驗證流程；若 in-app browser 工具不可用，使用 Edge headless 備援 |
| imagegen | 產生課程封面、單字卡、徽章與遊戲圖 | 已產生第一張課程封面 |
| verified-delivery | 啟動 DB/API/frontend 並實測 | 已用於目前實作流程 |
| Spreadsheets | 後續管理單字表、題庫、音效清單 | 規劃用於內容大量匯入 |
| Documents | 後續產出需求書、教案說明與家長報告模板 | 規劃用於產品文件與報告 |
| RunComfy Docker runner | 執行 Seedance 與 Image2.0 生成任務 | 已補上 Linux container 執行路徑 |

## Skill 使用紀律

FutureLight 的外掛與 skill 必須先讀規則再使用，尤其是會影響輸出品質或專案狀態的工具。

| 類別 | Skill / 外掛 | 使用時機 | 文件落點 |
| --- | --- | --- | --- |
| 專案代辦 | `workspace-todo-guardian` | 每次編輯、實作、規劃、啟動服務前 | `Todo/README.md`、目前 open roadmap |
| Port 管制 | `workspace-port-guard` | 啟動 localhost、改 Vite/Rust/Docker port、debug port 前 | `docs/環境與Port規劃.md` |
| Remotion | `remotion-best-practices` | 建立 React 程序化影片、字幕、週報影片 | `docs/AI協作製作流程.md`、`video/remotion` |
| HyperFrames | `hyperframes` / `hyperframes-cli` | HTML/GSAP 影片、社群短片、字幕與轉場 | `docs/AI協作製作流程.md`、`video/hyperframes` |
| Image2.0 | `gpt-image-2` / `gpt-image-edit` | 需要準確文字、版面、角色/圖卡/封面 | `assets/images`、`assets/asset_manifest.json` |
| Seedance | `seedance-v2` | 4-15 秒角色短動畫、口型、教學片段 | `assets/video/generated`、`assets/asset_manifest.json` |
| 文件產製 | `documents` | 需要 Word/PDF/家長報告正式文件時 | `docs` 或輸出文件目錄 |

目前 Remotion、HyperFrames、Image2.0/GPT Image 2、Seedance 相關 skill 已存在，不需要重新安裝；要做的是每次任務前明確讀取並把限制寫入任務規格。

## 自訂外掛

### futurelight-assets

位置：`C:\Users\USER\plugins\futurelight-assets`

用途：

- 檢查 `assets/asset_manifest.json`
- 檢查音效 ID 是否完整
- 檢查圖片與音效檔案是否存在
- 產出資源缺漏報告

指令：

```bash
node C:\Users\USER\plugins\futurelight-assets\scripts\check-assets.mjs C:\Users\USER\Desktop\work\futurelight
```

### futurelight-content-checker

位置：`C:\Users\USER\plugins\futurelight-content-checker`

用途：

- 檢查 `content/courses`
- 檢查課程是否有單元
- 檢查單元是否有學習步驟
- 檢查練習題是否有 prompt 與 correct_answer
- 發布前列出 blocker

指令：

```bash
node C:\Users\USER\plugins\futurelight-content-checker\scripts\check-content.mjs C:\Users\USER\Desktop\work\futurelight
```
