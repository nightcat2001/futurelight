# 學習播放功能設計

## 功能 1：學習步驟播放

孩子依順序完成單元內的學習步驟。

步驟類型：

- 單字。
- 句子。
- 故事段落。
- 聽力。
- 跟讀。

## 功能 2：音檔控制

每個有音檔的步驟可播放、暫停與重播。

規則：

- 音檔播放事件需記錄。
- 音檔失敗時可重試。
- 無音檔內容不顯示播放器。

## 功能 3：步驟完成紀錄

孩子完成每個步驟後，系統記錄進度。

規則：

- 不重複建立相同完成紀錄。
- 中途離開後可以從未完成步驟繼續。
- 完成最後步驟後更新單元學習狀態。

## 功能 4：跟讀練習入口

需要發音練習的步驟可提供錄音或跟讀入口。

初期規則：

- 初期可先做真實錄音/跟讀完成紀錄，不做語音評分，但不可用假按鈕或假完成狀態。
- 後續可加入語音辨識與發音評分。

## 功能 5：單元完成導向

單元學習完成後，導向下一個合理動作。

導向規則：

- 有練習題時導向練習遊戲。
- 沒有練習題時導向獎勵成就或課程詳情。
- 課程全部單元完成時，顯示課程完成提示。
