# 練習遊戲功能設計

## 功能 1：建立練習 session

孩子開始練習時，系統建立一次練習 session。

規則：

- session 綁定孩子、課程或單元。
- 題目順序可依系統設定固定或隨機。
- session 完成後不可再修改答案。

## 功能 2：答題互動

孩子依題型完成互動答題。

題型：

- 單選題。
- 圖片選擇。
- 聽音選答案。
- 配對。
- 排序。

## 功能 3：即時回饋

答題後立即顯示鼓勵或提示。

規則：

- 答對顯示鼓勵訊息。
- 答錯顯示正確提示，但避免挫折語氣。
- 可依題目設定是否允許再試一次。

## 功能 4：結果計算

練習完成後計算本次結果。

結果資料：

- 題目總數。
- 答對數。
- 答錯數。
- 答對率。
- 獲得星星。
- 建議複習項目。

## 功能 5：弱點回寫

答錯資料需回寫到孩子能力分析。

規則：

- 同一能力分類錯誤次數增加。
- 後續推薦可優先安排相關練習。
- 家長中心與學習進度可看到需要加強內容。
