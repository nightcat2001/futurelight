# 獎勵成就功能設計

## 功能 1：星星與點數

孩子完成學習或練習後可取得星星或點數。

規則：

- 完成單元可獲得基本星星。
- 練習表現佳可獲得額外星星。
- 獎勵計算需可追溯來源。

## 功能 2：徽章

孩子達成特定條件後取得徽章。

徽章條件範例：

- 第一次完成單元。
- 連續學習 3 天。
- 完成第一門課程。
- 練習全對。

## 功能 3：任務

系統提供短期任務引導孩子學習。

任務範例：

- 今天完成 1 個單元。
- 本週練習 3 次。
- 複習 5 個單字。

## 功能 4：獎勵展示

孩子可以查看已獲得與尚未獲得的獎勵。

規則：

- 已獲得徽章顯示取得時間。
- 未獲得徽章顯示取得條件。
- 隱藏型徽章可只顯示問號。

## 功能 5：防重複發放

不可重複獲得的獎勵需避免重複發放。

規則：

- 徽章以孩子與徽章 ID 做唯一限制。
- 點數型獎勵每次發放都要有事件紀錄。
- 系統重試時不可造成重複加分。
